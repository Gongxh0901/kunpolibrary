# KunpoLib
基于 Cocos Creator 3.0+ 的一套游戏框架，提供了一系列实用模块，帮助开发者快速构建高质量的游戏项目。
项目持续优化中，敬请期待~~~


## 安装kunpocc
项目已发布到 `npm`, 安装方法如下：
> 如果连不上npm, 可使用国内镜像 比如: 阿里
```bash
npm install kunpocc
```
